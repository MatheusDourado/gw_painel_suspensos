# GW Painel Suspensos

Painel administrativo para gestao de suspensos.

## Requisitos
- Node.js 18+ (ou 20+ recomendado)
- npm

## Instalacao
npm install

## Desenvolvimento
npm run dev

## Build e start
npm run build
npm run start

## Lint
npm run lint

## Variaveis de ambiente
Crie um arquivo .env com as variaveis necessarias.
